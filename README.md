# frequency.garden

Audio-reactive generative visual field. Feed it sound — from your mic, an audio file, or the built-in oscillator — and watch the frequencies bloom into particle structures.

Low frequencies grow large slow particles. Mids build medium structures. Highs scatter fine dust. The garden responds in real time.

---

**Stack:** Vite, vanilla JS, Canvas API, Web Audio API

**Try it:** https://frequency.pr0xy.dev

**Controls:**
- `m` — microphone input
- `o` — built-in oscillator (no mic required)
- `f` — load an audio file
- `s` — stop all
- `c` — clear particles

Adjust sensitivity, decay, and bloom intensity in the control bar.
